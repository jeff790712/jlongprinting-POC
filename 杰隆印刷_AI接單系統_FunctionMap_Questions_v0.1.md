## Part A：功能地圖（Function Map）

> 標示說明：✅ 本次 POC 包含 ／ 🔲 Phase 2 ／ ❓ 待確認是否列入

### A1 — 對話介面（網頁 Chatbot）

| # | 功能 | 子功能 | 說明 | 狀態 |
|---|------|--------|------|------|
| 1.1 | 對話入口 | 網頁嵌入框 | 可嵌入杰隆官網的對話視窗，RWD 支援桌機與手機 | ✅ |
| 1.2 | 對話入口 | 意圖識別 | 識別客戶來意（詢價 / 訂單追蹤 / 補印 / 閒聊 / 轉接） | ✅ |
| 1.3 | 詢價流程 | 開場引導 | 白話問候，建立 Session，判斷新/舊客 | ✅ |
| 1.4 | 詢價流程 | A 類欄位缺口偵測 | 即時追蹤 14 項欄位進度，依優先序追問 | ✅ |
| 1.5 | 詢價流程 | 材質智能引導 | 白話描述對應官網 8 種材質，不推薦目錄外材料 | ✅ |
| 1.6 | 詢價流程 | B 類條件欄位觸發 | 特定回答（成捲/燙金）自動追問條件欄位 | ✅ |
| 1.7 | 詢價流程 | 訂單草稿確認 | 彙整欄位摘要供客戶最終確認 | ✅ |
| 1.8 | 進度追蹤 | 收集進度顯示 | 對話側欄顯示已填/待填欄位狀態 | ✅ |
| 1.9 | 圖檔上傳 | 設計稿上傳 | 支援 JPG / PNG / PDF 上傳 | ✅ |
| 1.10 | 圖檔上傳 | AI 圖像分析 | LLM Vision 解析色數、特殊工藝，回報客戶確認 | ✅ |
| 1.11 | AI 模擬圖 | 情境圖生成 | 依欄位生成貼標容器情境圖 | ✅ |
| 1.12 | AI 模擬圖 | 對話內預覽 | 低解析度預覽（含浮水印）顯示於對話框 | ✅ |
| 1.13 | 舊客識別 | Email / 電話比對 | 比對客戶資料庫判斷新/舊客 | ✅ |
| 1.14 | 舊客識別 | 歷史訂單帶入 | 一年內舊客帶入舊欄位，只確認差異 | ✅ |
| 1.15 | 補印流程 | 補印舊款式 | 舊客補印同款，3 輪對話完成 | ✅ |
| 1.16 | 情緒偵測 | 負面情緒判斷 | 連續 2 次不滿語句觸發轉接 | ✅ |
| 1.17 | 情緒偵測 | 真人轉接通知 | 停止推進欄位，通知業務，保留對話 | ✅ |
| 1.18 | 情緒偵測 | 超規格轉接 | 材質/尺寸超範圍或客戶明確要求真人時轉接 | ✅ |
| 1.19 | 多語言 | 英文詢價 | 英文輸入自動處理，欄位對應中文訂單格式 | ✅ |
| 1.20 | LINE 頻道 | LINE 詢價入口 | 透過 LINE 官方帳號進入 Odin 工作流 | 🔲 Phase 2 |
